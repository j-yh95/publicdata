### 2. 카카오 로드뷰
![ezgif-3-c4712298c9c9](https://user-images.githubusercontent.com/30567909/96805333-5a4d0880-144c-11eb-828c-34e00677c423.gif)

1. 로드뷰 버튼을 누르면 도로 위에 이동 가능한 로드뷰 위치가 표시/제거됩니다.
2. 스카이뷰를 누르면 미니맵이 위성으로 표시/제거됩니다.
3. 경계를 누르면 법정동, 법정리 구분이 표시/제거됩니다.
4. 카카오/네이버를 누르면 현재 위치의 파노라마를 카카오/네이버에서 확인할 수 있습니다.
5. 우측 상단에는 마커를 제어하기 위한 버튼들입니다.
6. 미니맵에 마우스 우클릭시 해당 위치의 지번주소, 위/경도 값이 클립보드에 복사됩니다.
